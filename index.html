Rickest Rick

You will be given a random Rick number at the start of the game

There are four Ricks below. By clicking on a Rick you will add a specific amount of points to your Rick score.

You win the game by matching your score to Rick number, you lose the game if your total score goes above the Rick number.

The value of each Rick is hidden from you until you click on it

Each time when the game starts, the game will change the values of each Rick.
